# Documents

记录自己平时的一些简单文档

本文档使用`Markdown`撰写，并通过[MkDocs](https://www.mkdocs.org/)来管理，
你可以通过本地安装`mkdocs`命令行工具来实现本地浏览。

如下命令展示了如何在Ubuntu系统下通过`pip`命令安装`mkdocs`：

```shell
# 在使用pip之前，你可能需要先安装它
sudo apt install python3-pip
cd document
# 通过pip安装mkdocs
sudo -H pip3 install -r requirements.txt
# 启动服务，根据控制台输出打开浏览器
mkdocs serve
```
